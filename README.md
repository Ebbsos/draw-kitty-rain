Magic Draw - Kitty Rain

Proyecto en Python que usa la cámara, OpenCV y MediaPipe para dibujar con el dedo índice en tiempo real.
Si dibujas un corazón, se activa una animación especial de lluvia de gatitos.

Requisitos
Python 3.10 o superior
Cámara web
VS Code recomendado
Instalación
pip install -r requirements.txt
Ejecutar
python main.py
Controles
Dedo índice levantado: dibujar

main.py → código principal
hand_landmarker.task → modelo de detección de manos
kitty1.jpg a kitty8.png → imágenes para la animación
Tecnologías usadas
OpenCV
MediaPipe
NumPy
Características
Dibujo por gestos con la mano
Detección de corazones
Interfaz estilo scrapbook
Animación Kitty Rain
